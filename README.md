# GoLang

### Commands
Execute a go file:
```sh
go run main.go
```

Execute a go program:
```sh
go run *.go
```

Documentation:
```sh
go doc fmt
```

Check program Health:
```sh
go vet main.go
```

Build a Program:
```sh
go build main.go
```

Install a package:
```sh
go get -u github.com/go-sql-driver/mysql
```
